# `impact_functions/` (Calibrated direct damage and business interruption impact functions.)

This directory contains the impact functions that were calibrated and used to compute direct impacts and business interruption impacts from the different hazards:

- `river_floods.csv`
- `tropical_cyclones.csv`
- `storm_europe.csv`
- `tropical_cyclones_agriculture.csv`
- `FL_business_interruption.csv`
- `TC_business_interruption.csv`

Each files define impact functions to be loaded with CLIMADA.
